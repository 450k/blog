リポジトリを作る

githab pagesを利用するにはリポジトリの作成が必要です。 手順は以下の通り。

1 リポジトリの新規作成

githubにログインして画面右上、アイコン画像のすぐ左にある「＋」マークをクリックし、New repositoryをクリック。

2 リポジトリの名前を入力

3 完成 (内容の確認)

Markdowファイル作成時の設定Front Matterについて

2017, Nov 14

(記述間違によるコンパイルエラーが出て困ったのでメモ)
Front MatterとはYAMLやTOMLの記述で各ページのパラメータを設定するものです。
※使っているテンプレートによって異なるので注意が必要です。

+で区切る TOML Front Matterと
-で区切る YAML Front Matterがある。

TOML Front Matter

+++
parameter1 = "値"
parameter2 = true
+++

YAML Front Matter

---
parameter1: "値"
parameter2: true
---

どちらを使用しても問題ないが、パラメータと値をつなぐ記号が : と = で異なるので、 混在してしまうとエラーの原因となるので注意。

今回はYAML Formatter で統一することにしました。
